About

Home Assistant is an open source home automation package powered by a worldwide community of tinkerers and DIY enthusiasts.

Documentation is extensive, but not easy to navigate - hence this unofficial index.

All the links on this page are to official documentation.

Note: this page does not include links to vendor- and device-specific implementations. Home Assistant have their own index here.

To request changes or additions, please use the GitHub discussions page.
